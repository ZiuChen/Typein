# *TypeIn*

*唤醒-键入-激活, 键入文字即可触达任何功能, 让一切操作变得简单且高效。*

*TypeIn*是一个运行在*Microsoft Edge*上的Add-on拓展，它可以让你在保持专注的情况下，高效完成各种操作。

* 即时计算/翻译/搜索/备忘，用完即走
* 网页快开、创建/解析二维码、速记

## 后续计划

* 将功能模块化，支持用户编写`JS`代码以自定义任何功能
* 建立社区，用户可以在社区内分享自己编写的`Action`
